# Public Run Spec Schema

Each scan/research session should include:

```text
reports/sessions/<session-id>/driverstore-run-spec-public.md
```

Future modules may use:

```text
reports/sessions/<session-id>/<module>-run-spec-public.md
```

Required sections:

- Session metadata
- Windows context
- Machine context
- Modules covered
- Scan summary
- Research summary
- Execution summary
- Public case notes
- Privacy checklist

The run spec must not include private mappings, raw local inventory, usernames,
serial numbers, local paths, or exact private identifiers.
