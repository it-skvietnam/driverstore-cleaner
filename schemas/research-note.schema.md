# Research Note Schema

Research notes are append-only public records under:

```text
research-notes/YYYY-MM-DD-<session-id>-<agent-tool>.md
```

Required fields:

```text
Entry ID
Date
Session ID
Agent tool
Agent model
Operator
Run mode
Windows generation
Windows release
OEM
Machine family
Model line
Public files
What was done
What was learned
Privacy review
```

Do not delete or rewrite old notes. Add correction notes when needed.
